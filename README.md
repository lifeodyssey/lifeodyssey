## 👋 About Me

I'm a backend-focused software engineer at **Thoughtworks**, working around backend systems, cloud infrastructure, DevOps automation, and practical AI engineering.

I care about systems that survive real production pressure: clear boundaries, boring reliability, observable behavior, and workflows that help teams move faster without making the codebase harder to change.

Recently, I'm spending more energy on **Agentic AI**, **RAG**, **coding agents**, **context engineering**, and how AI can become a dependable part of everyday software delivery instead of just a demo.

- 🧱 **Backend:** APIs, service design, Spring Boot, FastAPI
- ☁️ **Cloud / Infra:** AWS, Terraform, Terragrunt, Docker, CI/CD
- 🤖 **AI Engineering:** agent workflows, RAG, coding agents, tool/plugin systems
- ✍️ **Writing:** long-form notes about AI engineering, backend, cloud, and developer tooling
- 🌱 **Principle:** build useful things, then make them maintainable
